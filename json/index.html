The internet is dynamic.

Most often, you’ll visit a website and see a range of content. Now, you leave and come back to that site an hour later. The content is different. One of the most important things on the internet is design, but the other is content. Content needs to change in order to keep users coming back to your page.

API

Application Programming Interface, or API is a tool used to specify how programs should work together. Many sties use API’s as a way to allow developers to share their information. API’s can be used in many ways, XML, JSON etc. Today we’ll be focusing on JSON

What is JSON?

Javascript Object Notation is the name of one of the most widely used formats included in API’s. It’s written in pure Javascript and meant to be easily read by computers and humans.

Javascript Object

Everything in Javascript is an object. We interact with objects throughout javascript. Variables are objects, as well in jQuery, referencing an element via the $(‘name’) method is selecting an object.

JSON can be a single object that contains many values or an array that contains much more information. JSON Data is written as name/value pairs, which evaluate to simple variables.

JSON Syntax

Name/Value Pairs

Data is required to stated with a name and a value in quotes, seperated by a colon. The standard syntax for writing names is camel case with no spaces ie ‘firstname’.

// JSON
"firstName": "Drew"

// Evaluates to
var firstName = "Drew"

JSON Objects

JSON objects are written inside curly braces and can contain multiple name/value pairs

// JSON
{ "firstName": "Drew", "lastName": "Minns", "favouriteFood": "Burritos"}

// Evaluates to
var firstName = "Drew";
var lastName = "Minns";
var favouriteFood = "Burritos";

JSON Array

Arrays can be used to define multiple objects. They are defined with a name and square brackets.

// JSON
"people": [
    {"firstName": "Drew", "lastName": "Minns", "favouriteFood": "Burritos"}
    {"firstName": "Alice", "lastName": "Cooper", "favouriteFood": "Chicken"}
    {"firstName": "Ozzy", "lastName": "Osborne", "favouriteFood": "Bat"}
]

The above array is named ‘people’ and contains three objects, each containing values for firstName, lastName and favouriteFood.

Traversing JSON

In order to access values within JSON, we need to recognize that JSON has a heirarchy and different levels. Take the following example.

// JSON
var myData = {
    "firstName": "Drew",
    "lastName": "Minns",
    "details": [
        {
        "favouriteFood": "Burritos",
        "age": "29"
        }
    ]
    }

The firstName, lastName values are on the root level. favouriteFood and age however are within details. JSON can be many levels deep. To traverse JSON, you need to build a path to access those objects. The basic example below will show how to access the objects in the above JSON feed by their names and apply the values to variables.

// Take the values of firstName and lastName and save them to a variable called name
var name = myData.firstName + myData.lastName;

// Access the value of favouriteFood and apply it to the variable food. Note how we have to
// reference the name of the details array to get details inside
var food = myData.details.favouriteFood;

// Access the age object inside details
var age = myData.details.age;

Numbering

When counting in Javascript, it is important to recogize that counting doesn’t start at 1. It starts at zero and goes up from there. If you are referencing the first object, you need to say 0 instead of 1.

var jsonArray = {
    "data": [
        {"item": "item1"},
        {"item": "item2"},
        {"item": "item3"}
    ]
}

So take the example JSON above. In the array called Data, we have three objects that contain items called ‘item’ with different values for each. To access those values we need to reference each object individually. The syntax to do requires calling what item you would like. For example, if we want the contents of the second object in the data array, we would refer to it as [1]. Note the square brackets around the number.

// Example to get the second object.
var secondObject = jsonArray.data[1].item;

// Which will output 'item2'

Outputting Data

To output data, simply use jQuery (or plain javascript) to reference the JSON object and apply it to an object.

// Append to body
$(secondObject).appendTo("body");

External Data

jQuery offers a useful method for accessing data that is not stored in the same file as the rest of your code. the getJSON method allows an external source of JSON be pulled directly into your code and be interacted with.

// feed.json
{
    "users": [
        {
            "firstName": "Drew",
            "lastName": "Minns",
            "age": "29"
        },
        {
            "firstName": "David",
            "lastName": "Bowie",
            "age": "67"
        },
        {
            "firstName": "Peter",
            "lastName": "Gabriel",
            "age": "64"
        }
    ]
}

// app.js

$(document).ready(function() {

    // Call the json by the filename and path and apply a name to the information (data)
    $.getJSON('feed.json', function(data) {

        // Get First User information
        var firstUserName = data.users[0].firstName;

        // Apply to HTML
        $("body").append(firstUserName);

    });

});
