Github

When working within a team, keeping code up to date within all members of a team is important. As well, having a central place to keep code protected and showcased is imperative.

Github is an online service to allow code to be managed and showcased. The system it uses is called Git, which was developed by Linus Torvalds, the creator of the operating system Linux in 2005.

Terminology

Profile

Your user page. Displays your repositories and activities on the site.

Repository

Most basically, a project. Think of it as a folder on your computer that contains your code for a project. The repository contains all the project files and keeps the history of all the files.

Commit

A marker of file changes. Essentially a record of an individual save that includes the time, date and who made what changes. Commits contain a message that states what changes were made.

Branch

A version of the repository that contains seperate code. Typically used to add features which will not affect the ‘master’ branch (the main branch) and can be merged back into the ‘master’ branch when completed.

Push

Sends your committed changes from your computer (or local machine) to the repository. Once your commits are pushed to the repository, another user can pull the changes to their computer.

Pull

Downloading the latest changes to the repository to your computer.

Clone

A copy of the files included in the repository or the act of obtaining those files.

Floats

Float is a widely used CSS positioning property. They are designed to be used on images in blocks to text, to allow text to wrap around images, much like in print design. Applying a float in a webpage tells content to flow around the object.

Beyond images, the float property can be used to layout objects on a page.
You can define float: left; and float: right; to tell an object to position itself accordingly.

Think of HTML elements as 100% width blocks. By default they will fill the screen horizontally. With floats and a width, you can tell an element to fill next to each other.

Clearing A Float

The sister element to the Float property is Clear. Using clear, you can tell an object to again respect the block element structure. Using clear:both; on an element will stop the object from breaking the natural flow of a webpage.
